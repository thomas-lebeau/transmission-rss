## Getting started

In your directory, run:

```
curl -fsSL https://github.com/thomas-lebeau/quickie/archive/master.tar.gz | tar xz --strip-components=1
```

## Deploy to surge

Add the folowing secret to the Deloy Github action:

- `SURGE_TOKEN` (get it with `surge token`)
- `SURGE_LOGIN`
